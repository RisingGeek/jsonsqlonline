Step 2: Query JSON Object


  • Enter a valid json string
  • The parsed JSON object is the database
  • Use sql like queries on ths json object
  • select name from country returns a list of country names
  • select name, dignitaries.president from country returns a list of country names and their presidents
  • select *
    from country
    where this.name === 'India'
    returns all objects in the country array that have name == 'India'
  • select *
    from country.0.dignitaries
    returns all properties in the dignitaries object located in the first object of the country array
